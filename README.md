# 🚀 GitHub Analytics Pro

GitHub Analytics Pro is a modern React-based web application that allows users to search for any GitHub profile and view detailed insights, including profile statistics and top repositories in a clean, responsive dashboard.

---

## ✨ Features

- 🔍 Search GitHub users by username
- 👤 View user profile information
  - Profile picture
  - Name and bio
  - Followers & Following
  - Public repositories
- ⭐ Display top starred repositories
- 🍴 Show repository forks
- 💻 Display primary programming language
- 🔗 Direct links to GitHub repositories
- 🌙 Professional dark-themed responsive UI

---

## 🛠️ Tech Stack

- React.js
- JavaScript (ES6)
- CSS3
- GitHub REST API

## 🚀 Getting Started

### Clone the repository

```bash
git clone https://github.com/your-username/github-analytics-pro.git
```

### Navigate to the project

```bash
cd github-analytics-pro/frontend
```

### Install dependencies

```bash
npm install
```

### Start the development server

```bash
npm start
```

The application will run at:

```
[http://localhost:3000](http://localhost:3000/GitHub-Analytics-Pro)
```

## 🌐 Deployment
