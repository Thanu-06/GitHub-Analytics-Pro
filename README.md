# GitHub Analytics Pro

A React dashboard that fetches GitHub user data and displays profile insights and top repositories with a professional UI.

## Live Demo
Planned GitHub Pages URL: https://Thanu-06.github.io/GitHub-Analytics-Pro

## Features
- GitHub user search
- User profile display with avatar, followers, following, and public repo count
- Top starred repositories with stars, forks, language, and GitHub link
- Resume-ready dark dashboard theme

## Deployment
This project is configured to deploy automatically to GitHub Pages from the `frontend/build` folder on every push to `master`.

## Run locally
```bash
cd frontend
npm install
npm start
```
